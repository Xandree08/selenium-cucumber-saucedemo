# selenium-cucumber-saucedemo
Projeto de automação de testes E2E para a aplicação web SauceDemo, desenvolvido com Java, Selenium WebDriver e Cucumber (BDD). O objetivo é validar fluxos críticos como login, carrinho e checkout, aplicando boas práticas como Page Object Model, Gherkin, esperas explícitas, relatórios de testes e estrutura escalável para portfólio QA Automation.
